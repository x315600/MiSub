# MiSub 代理插件使用指南

## 📖 项目概述

MiSub代理插件是MiSub订阅转换器的可选插件，提供VLESS和Trojan协议的代理服务。基于Cloudflare全球边缘网络，安全可靠、快速稳定。

### 🎯 主要特性
- **🔌 插件设计**: 可选安装，不影响MiSub主功能
- **🌐 双协议**: 支持VLESS和Trojan协议
- **⚡ 全球加速**: Cloudflare边缘网络支持
- **🔐 安全加密**: TLS强制加密，安全可靠
- **📱 简单易用**: 一键配置，实时反馈

## 🚀 快速开始

### 1. 启用代理服务
1. 登录MiSub管理后台
2. 进入设置页面
3. 找到"代理服务插件"选项
4. 打开"启用代理服务"开关

### 2. 基础配置
```javascript
必填配置项：
- UUID: 用户唯一标识符 (格式: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx)
- 代理IP: 代理服务器地址 (格式: ip:端口 或 域名:端口)
- 订阅路径: 访问链接的路径 (格式: link, proxy_sub等)

可选配置项：
- Trojan协议: 可选择开启
- 优选IP: 多个备用IP地址
```

### 3. 获取订阅链接
配置完成后，在页面复制订阅链接：
```
https://your-domain.com/proxy/你的订阅路径
```

### 4. 客户端配置
**V2rayN:**
1. 复制订阅链接
2. 在V2rayN中添加订阅
3. 更新订阅并选择节点

**Clash:**
1. 复制订阅链接
2. 在Clash中添加订阅源
3. 下载配置文件

**Shadowrocket:**
1. 复制订阅链接
2. 在Shadowrocket中添加订阅
3. 启用代理

## ⚙️ 配置说明

### 基础配置
| 配置项 | 说明 | 示例 |
|--------|------|------|
| **启用代理服务** | 总开关，关闭后插件完全不可用 | ✅ 开启 |
| **UUID** | 用户唯一标识符，必须正确填写 | `123e4567-e89b-12d3-a456-426614174000` |
| **代理IP** | 代理服务器地址 | `proxy.example.com:443` |
| **订阅路径** | 订阅链接的访问路径 | `link` |

### 协议选择
- **VLESS**: 默认启用，无需配置，自动支持
- **Trojan**: 可选开启，适合需要伪装成HTTPS的场景

### 优选IP配置
```
格式：IP:端口#备注名称

示例：
1.1.1.1:443#Cloudflare
8.8.8.8:443#Google
proxy.example.com:443#自定义代理
[2001:db8::1]:443#IPv6地址
```

**支持的格式：**
- IPv4地址: `192.168.1.1:443`
- 域名: `proxy.example.com:443`
- IPv6地址: `[2001:db8::1]:443`
- 带备注: `IP:端口#备注名称`

## 🔧 常见问题

### Q: 为什么代理服务无法启用？
A: 检查必填项是否正确填写：
- UUID格式是否正确
- 代理IP格式是否有效
- 订阅路径是否合法

### Q: 订阅链接无法访问怎么办？
A: 确认以下几点：
- 代理服务开关已打开
- 订阅链接格式正确
- 域名已正确解析

### Q: 客户端连接失败怎么解决？
A: 尝试以下方法：
- 检查客户端配置的UUID是否正确
- 验证代理IP地址和端口
- 尝试使用不同的优选IP
- 检查网络连接是否正常

### Q: 如何重置配置？
A: 在代理设置页面重新填写所有配置项即可，系统会自动保存新的配置。
